his project is a simple FastAPI application that manages a list of courses.
It demonstrates how to:
- define Pydantic models
- store in‑memory data
- build REST API endpoints
- retrieve all courses
- retrieve a single course by ID
It’s a great foundation for learning FastAPI and building more advanced APIs.

Tech Stack
- Python 3.10+
- FastAPI
- Uvicorn
- Pydantic
